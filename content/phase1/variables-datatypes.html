Think of variables as labeled storage boxes in your computer's memory. Each box has a name (variable name) and can store different types of data. Just like physical storage boxes, you can put things in, take things out, and change what's inside.

Python Variables in Action 
# Numbers
age = 25                    # Integer (whole number)
height = 5.8               # Float (decimal number)

# Text
name = "Alice"             # String (text)
is_student = True          # Boolean (True/False)

# You can check the type of any variable
print(type(age))           # 
print(type(height))        # 
print(type(name))          # 
print(type(is_student))    # 

# Using variables in sentences
print(f"Hello! My name is {name}")
print(f"I am {age} years old and {height} feet tall")
print(f"Am I a student? {is_student}")

üîç Data Type Deep Dive

üî¢ Integers (int)

What they are: Whole numbers, positive or negative, including zero

When to use: Counting things, ages, scores, array indices

Memory efficient: Take up less space than floats

Try Different Integer Values:
# Integer examples
positive_num = 42
negative_num = -17
zero = 0
large_num = 1000000

# Math with integers
result = positive_num + negative_num  # 25
print(f"42 + (-17) = {result}")

üéØ Floats (float)

What they are: Numbers with decimal points

When to use: Measurements, percentages, scientific calculations

Precision: Can represent very small or very large numbers

Explore Float Precision:
# Float examples
pi = 3.14159
temperature = -2.5
percentage = 85.7
scientific = 1.23e-4  # Scientific notation

# Precision demonstration
result = 0.1 + 0.2
print(f"0.1 + 0.2 = {result}")  # Might not be exactly 0.3!

üìù Strings (str)

What they are: Text data enclosed in quotes

When to use: Names, messages, file paths, any text

Flexible: Can contain letters, numbers, symbols, emojis

String Playground:
# String examples
single_quotes = 'Hello, World!'
double_quotes = "Python is awesome!"
multi_line = """This is a
multi-line string"""

# String operations
name = "Alice"
greeting = f"Hello, {name}!"  # f-string formatting
length = len(name)  # String length
upper_name = name.upper()  # Convert to uppercase

‚úÖ Booleans (bool)

What they are: True or False values only

When to use: Decision making, flags, conditions

Foundation: Essential for if statements and loops

Boolean Logic Tester:
# Boolean examples
is_sunny = True
is_raining = False
has_umbrella = True

# Boolean logic
can_go_outside = is_sunny or (is_raining and has_umbrella)
print(f"Can go outside? {can_go_outside}")

# Comparison results are booleans
age = 25
is_adult = age >= 18  # True
is_teenager = 13 <= age <= 19  # False

üìù Variable Naming Rules & Best Practices

‚úÖ Valid Names

  • user_name - Snake case (recommended)
  • firstName - Camel case (OK)
  • _private_var - Starting with underscore
  • count1 - Letters and numbers
  • CONSTANT_VALUE - All caps for constants

‚ùå Invalid Names

  • 2nd_grade - Can't start with number
  • user-name - No hyphens allowed
  • for - Python keyword
  • my var - No spaces allowed
  • user@name - No special characters
