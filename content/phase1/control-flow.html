🏗️ Conditional Logic Patterns

🎯 Simple If Statement

When to use: When you want to execute code only if a condition is true.

Pattern: "If this condition is true, then do something."

You are eligible to vote! 🗳️
# Simple If Example
age = 18

if age >= 18:
    print("You are eligible to vote! 🗳️")
    
# If the condition is false, nothing happens
# The program just continues to the next line

⚖️ If-Else Statement

When to use: When you have exactly two possible outcomes.

Pattern: "If this condition is true, do A, otherwise do B."

42 is even! ✅
# If-Else Example
number = 42

if number % 2 == 0:
    print(f"{number} is even! ✅")
else:
    print(f"{number} is odd! 🔸")
    
# One of these two blocks will ALWAYS execute

🎪 If-Elif-Else Chain

When to use: When you have multiple mutually exclusive conditions.

Pattern: "If A, do this; else if B, do that; else if C, do another thing; otherwise, do default."

Score: 85 → Grade: B 📚
# If-Elif-Else Example
score = 85

if score >= 90:
    grade = "A"
    emoji = "🏆"
elif score >= 80:
    grade = "B" 
    emoji = "📚"
elif score >= 70:
    grade = "C"
    emoji = "📖"
elif score >= 60:
    grade = "D"
    emoji = "📝"
else:
    grade = "F"
    emoji = "😰"

print(f"Score: {score} → Grade: {grade} {emoji}")

🪆 Nested Conditions

When to use: When one condition depends on another being true first.

Pattern: "If A is true, then check if B is true, and if so, do something."

Perfect weather for outdoor activities! 🌞
# Nested Conditions Example
temperature = 75
weather = "sunny"

if temperature > 60:  # First condition: warm enough
    if weather == "sunny":
        activity = "Perfect weather for outdoor activities! 🌞"
    elif weather == "cloudy":
        activity = "Good weather, maybe bring a light jacket 🧥"
    else:
        activity = "Warm but not ideal weather ⛅"
else:  # Temperature <= 60
    if weather == "snowy":
        activity = "Great day for hot cocoa and staying warm! ❄️"
    else:
        activity = "Too cold for outdoor activities 🥶"

print(activity)

🧠 Logical Operators (and, or, not)

When to use: When you need to combine multiple conditions.

Patterns: "If A AND B", "If A OR B", "If NOT A"

🎮 Logical Operator Playground
# Logical Operators Example
age = 20
has_license = True
has_car = True

# AND: Both conditions must be true
if age >= 18 and has_license:
    print("✅ Can drive legally")

# OR: At least one condition must be true  
if has_license or age >= 21:
    print("✅ Can rent a car from some companies")

# NOT: Reverses the condition
if not has_car:
    print("🚌 Might need public transportation")

# Complex combinations
if age >= 18 and has_license and has_car:
    print("🚗 Ready for a road trip!")
elif age >= 18 and has_license:
    print("🚕 Can drive but might need to borrow/rent a car")
else:
    print("🚌 Not ready to drive yet")

# Choose Your Own Adventure - Python Implementation
def adventure_game():
    print("Welcome to CodeMaster Academy!")
    print("You discover a mysterious USB drive...")
    
    choice = input("What do you do? (investigate/report/ignore): ").lower()
    
    if choice == "investigate":
        print("You plug in the USB drive...")
        next_choice = input("Footsteps approach! (hide/run/stay): ").lower()
        
        if next_choice == "hide":
            print("🙈 You hide the USB drive and act innocent.")
            print("Ending: The Mystery Student")
        elif next_choice == "run":
            print("🏃‍♂️ You copy the files and escape!")
            print("Ending: The Code Thief")
        else:
            print("💪 You stay and meet the mysterious figure...")
            print("Ending: The Brave Confronter")
            
    elif choice == "report":
        print("You report the USB drive to Professor Python...")
        teacher_response = input("Accept his challenge? (accept/decline/ask): ").lower()
        
        if teacher_response == "accept":
            print("✅ You accept the advanced challenges!")
            print("Ending: The Honor Student")
        elif teacher_response == "decline":
            print("❌ You politely decline for now.")
            print("Ending: The Cautious Learner")
        else:
            print("❓ You ask to work with a partner.")
            print("Ending: The Collaborative Coder")
    
    else:  # ignore
        print("🚶‍♂️ You ignore the USB drive and focus on class.")
        print("Ending: The Focused Student")

# Every choice creates a different path using if/elif/else!
