🍪 The Cookie Cutter Analogy

🍪

Class

The cookie cutter (blueprint)

→
🍪 🍪 🍪

Objects

The actual cookies (instances)

Understanding the Analogy

  • Class: The cookie cutter - defines shape, size, and properties
  • Object: The actual cookie - created from the class blueprint
  • Attributes: Cookie properties (flavor, size, decorations)
  • Methods: Cookie actions (bake, decorate, eat)

Define the Class

2
Add the Constructor (__init__)
class Student:
    def __init__(self, name, age, major):
        # Constructor: initializes new objects
        self.name = name      # Instance attribute
        self.age = age
        self.major = major
        self.grades = []      # Empty list for grades
__init__ is a special method that runs when creating a new object. It sets up the initial state of the object.
3
Add Methods (Functions)
class Student:
    def __init__(self, name, age, major):
        self.name = name
        self.age = age
        self.major = major
        self.grades = []
    
    def add_grade(self, grade):
        """Add a grade to the student's record"""
        self.grades.append(grade)
    
    def get_average(self):
        """Calculate the student's average grade"""
        if not self.grades:
            return 0
        return sum(self.grades) / len(self.grades)
4
Add String Representation
class Student:
    def __init__(self, name, age, major):
        self.name = name
        self.age = age
        self.major = major
        self.grades = []
    
    def add_grade(self, grade):
        self.grades.append(grade)
    
    def get_average(self):
        if not self.grades:
            return 0
        return sum(self.grades) / len(self.grades)
    
    def __str__(self):
        """String representation of the student"""
        return f"Student: {self.name}, Major: {self.major}"
5
Create and Use Objects
# Creating objects (instances)
alice = Student("Alice", 20, "Computer Science")
bob = Student("Bob", 19, "Mathematics")

# Using methods
alice.add_grade(95)
alice.add_grade(87)
alice.add_grade(92)

print(alice)  # Output: Student: Alice, Major: Computer Science
print(f"Alice's average: {alice.get_average()}")  # Output: 91.33
